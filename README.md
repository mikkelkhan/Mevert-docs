What is Mevert?

Mevert comes from "Me" + "Vert" - representing everyone: Introverts, Extroverts, and Ambiverts.
Mevert is a social platform that helps people find companions for real-world activities and events.
Sometimes you want to go to a movie, stand-up comedy show, concert, cafe, or any event, but you don’t want to go alone. With Mevert, users can create an event and share their plan.
People who are interested in the same activity or location can discover the event and connect to join.
Mevert makes it easier to meet new people, share experiences, and explore places together.

## Tech Stack
Backend: Python / Flask  
Frontend: HTML, CSS, JavaScript  
Database: PostgreSQL                                                                                                                                                     
Deployment: Render
